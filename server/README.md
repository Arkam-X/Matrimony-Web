## Folders architecture and Funstions
models → database schemas

controllers → request handling

routes → API endpoints

middleware → auth, errors, security

services → business logic (tokens, email, etc.)

utils → reusable helpers

## Backend Dependencies :-
express, mongodb, dotenv, cors, bcryptjs, jsonwebtoken, cookie-parser, express-calidator, helmet, morgan, nodemon
